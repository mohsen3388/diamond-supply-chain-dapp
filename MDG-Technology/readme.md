# Chapter 6 – MDG Technology and UML Profile Support

This folder contains the Enterprise Architect MDG Technology artifacts developed to support the proposed model-driven framework.

It includes:

- UML Profile for the Collaborative Process Definition Metamodel (CPDM)
- UML Profile for the Collaborative Process Execution Metamodel (CPEM)
- Enterprise Architect toolbox definitions
- MDG Technology configuration files
- Enterprise Architect project files

These artifacts support the tool-based realization of the theoretical framework presented in Chapter 6 of the thesis.
