# A Model-Driven and Blockchain-Supported Approach to Improve the Definition and Execution of Collaborative Supply Chain Processes

## Overview

**"A Model-Driven and Blockchain-Supported Approach to Improve the Definition and Execution of Collaborative Supply Chain Processes"**

## Research Contribution

This research proposes a Model-Driven Engineering (MDE) framework that integrates blockchain technology with collaborative supply chain process management.

The proposal is based on:

1. A Collaborative Process Definition Metamodel (CPDM)
2. A Collaborative Process Execution Metamodel (CPEM)
3. Model-to-Model (M2M) transformation rules
4. Model-to-Text (M2T) transformation rules
5. UML Profiles and Enterprise Architect MDG Technology
6. Automated generation of blockchain-oriented execution artifacts
7. Smart contract deployment and validation on Ethereum-based environments

The framework is validated through a diamond supply chain case study involving multiple collaborative stakeholders, blockchain traceability mechanisms, and smart contract execution.
