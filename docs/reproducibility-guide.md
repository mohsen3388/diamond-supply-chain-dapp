# Reproducibility Guide

This repository contains the main technical artifacts associated with the thesis.

## Chapter 6 Artifacts

The MDG Technology and UML Profile files can be imported into Enterprise Architect to create CPDM and CPEM models.

## Chapter 7 Artifacts

The blockchain prototype contains Solidity smart contracts, test cases, deployment scripts, and the DApp implementation used for the experimental validation.

## Suggested Reproduction Process

1. Import the MDG Technology into Enterprise Architect.
2. Create a CPDM model using the provided UML Profile.
3. Apply the transformation rules to obtain the CPEM model.
4. Generate Solidity artifacts from the execution model.
5. Run the smart contract tests using Hardhat.
6. Deploy and execute the prototype DApp.
